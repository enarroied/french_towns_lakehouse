Population by Department

Population trends for French departments, 1968 - 2023

This dashboard was generated by OpenCode (Pickle model). The user provided the SQL query and general guidance.

It covers 100 departments — the 96 metropolitan departments plus the four overseas départements d’outre-mer (Guadeloupe, Martinique, Guyane, La Réunion). Mayotte, Saint-Pierre-et-Miquelon, Saint-Martin, Wallis-et-Futuna, Terres australes et antarctiques françaises, and Clipperton are excluded because INSEE population data does not cover them.
